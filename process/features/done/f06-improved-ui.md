# Feature description for feature F06
## F06 — Improved UI: progressive disclosure
**Priority**: High
**Done:** yes
**Tests Written:** yes
**Test Passing:** yes
**Description**: Replace the current static graph display with a step-by-step interactive textbook experience. The forward pass lesson starts with the equation, then steps through each node one at a time via a Next button. Each step highlights the active node in the graph and shows a plain-English explanation of what just computed (e.g. "len_term = length × w_len = 5.0 × 0.5 = 2.5"). The visualizer gains a `highlighted` parameter to color one node differently. The lesson uses a step counter driven by a Next button.
